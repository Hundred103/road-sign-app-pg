# Assets

This folder contains static assets for the Road Sign Learning App.

## Structure

```
assets/
├── signs/          # Road sign images
│   ├── warning/    # Warning signs (A-series)
│   ├── prohibition/# Prohibition signs (B-series)
│   ├── mandatory/  # Mandatory signs (C-series)
│   ├── information/# Information signs (D-series)
│   └── priority/   # Priority signs
├── sounds/         # Audio files
│   ├── correct.mp3
│   ├── incorrect.mp3
│   └── background/
└── images/         # UI images and icons
```

## Image Sources

- PixaBay (stock images)
- Wikimedia Commons

## Guidelines

- Road sign images should be in PNG format with transparent background
- Recommended size: 256x256 pixels
- Sound files should be in MP3 format
- Keep file sizes optimized for web
